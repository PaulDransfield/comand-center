
```

```

Order matters only for readability — Fortnox treats the scope list as a set, not a sequence. But keeping the original 5 first and the new 5 after makes the diff easy to review.

The 10 scopes and what each unlocks:

| Scope | What it unlocks for CommandCenter |
|---|---|
| `bookkeeping` | Vouchers / general ledger entries — the core financial data |
| `invoice` | Sales invoices |
| `supplierinvoice` | Supplier invoices — for AP analytics |
| `salary` | Salary transactions — for Lön activation later |
| `companyinformation` | Basic company info, fiscal periods |
| `costcenter` | Cost center / department dimensions on transactions — for dept_metrics |
| `customer` | Customer master records — for AR analytics, customer name lookups |
| `supplier` | Supplier master records — for supplier price-creep analytics |
| `timereport` | Attendance and scheduling time — pairs with `salary` for full Lön |
| `article` | Article master data — for future item-level analysis |

Update any inline comment near the scope string to reflect the new list. If there's a comment explaining what each scope is for, expand it to cover the new ones using the table above. If there's no comment, add a brief one referencing this prompt by date.
